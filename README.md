<h1 align="center">Hrank</h1>
<p align="center">Ranking table app</p>

## Usage

To run the application java at least java 17 is needed.

Execute the jar with the following command:

```bash
java -jar hrank-0.1.jar
```

Open the browser and enter at `localhost:8080` for scoreboard view. For admin pannel enter `localhost:8080/admin`.
